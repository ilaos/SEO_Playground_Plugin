## Installation

### From WordPress Admin
1. Upload the plugin to `/wp-content/plugins/almaseo-seo-playground/`
2. Activate through the Plugins screen
3. The Setup Wizard will guide you through initial configuration
4. Navigate to **SEO Playground** in the admin sidebar

### Migrating from Another Plugin
1. Go to **SEO Playground → Import SEO**
2. Select your current plugin (Yoast, Rank Math, or AIOSEO)
3. Use dry-run mode to preview what will be imported
4. Click Import to migrate all SEO data

## Configuration

### Quick Start
1. Complete the Setup Wizard (runs on first activation)
2. Set your Search Appearance templates for titles and descriptions
3. Configure sitemaps under SEO Playground → Sitemaps
4. Add verification codes for Google, Bing, etc.
5. Optionally connect to AlmaSEO dashboard for AI features
